# PRODUCTION-READY SCORECARD

## Purpose
Đo mức sẵn sàng production theo thang điểm 100.

## Categories
1. Playability (20)
2. Stability (20)
3. Network Reliability (15)
4. Content Readiness (15)
5. UX/Accessibility (10)
6. Performance (10)
7. QA Process Maturity (10)

## Scoring Rubric
- 90–100: Production Ready
- 75–89: Near Ready (targeted fixes)
- 60–74: At Risk (major gaps)
- <60: Not Ready

## Mandatory Minimums
- Playability >= 16/20
- Stability >= 16/20
- Network >= 12/15
- Blocker bugs = 0
