# VERTICAL SLICE PRODUCTION BIBLE (12 WEEKS)

## Goal
Deliver a publisher-ready vertical slice demonstrating:
- Core co-op identity
- Signature mechanics quality
- Art/audio polish target
- Stable performance and networking

## Scope
- 2 playable chapters (condensed)
- 1 full boss encounter
- 1 narrative arc complete (setup -> conflict -> resolution beat)
- 90–120 min polished gameplay

---
## WEEK-BY-WEEK PLAN

### Week 1 — Alignment & Lock
- Lock slice scope and kill-list
- Finalize level blockout ownership
- Confirm KPIs and Definition of Done

Deliverables:
- Scope lock doc
- Team allocation grid
- Risk register v1

### Week 2 — Core Loop Stability
- Movement + interaction baseline
- Session flow create/join/reconnect
- Initial split-screen behavior

Deliverables:
- Playable graybox loop v0
- Net smoke test report

### Week 3 — Ability Set Integration
- LightPulse / ShadowAnchor full loop
- Ability feedback VFX/SFX pass v0
- Puzzle state replication hooks

Deliverables:
- 5 puzzle rooms playable
- Ability reliability test sheet

### Week 4 — Encounter Layer v0
- Dual-shield enemies
- Co-op finisher prototype
- Encounter pacing pass first half

Deliverables:
- Combat room pack v0
- TTK and readability metrics

### Week 5 — Narrative Implementation v0
- Dialogue triggers integrated
- In-game cinematic handoffs setup
- Memory fragment beat #1 and #2

Deliverables:
- Narrative flow passable start-to-mid
- Subtitle pipeline test

### Week 6 — Boss Production v0
- Boss phase state machine playable
- Telegraph/readability pass
- Fail/retry loop tuning

Deliverables:
- End-to-end boss encounter v0
- QA defect report and priorities

### Week 7 — Art Pass 1
- Environment art uplift for both chapters
- Character material and silhouette polish
- Lighting language pass

Deliverables:
- Visual target comparison board
- Art quality delta report

### Week 8 — Audio Pass 1
- Adaptive score tiers integrated
- Critical SFX clarity pass
- VO implementation first pass

Deliverables:
- Audio mix profile A/B
- Intelligibility metrics

### Week 9 — Performance & Stability Sprint
- CPU/GPU optimization
- Streaming memory budget tuning
- Network desync fixes

Deliverables:
- Perf report against target hardware
- Crash triage burn-down chart

### Week 10 — Playtest & Balance
- External blind playtest cohort
- Puzzle difficulty rebalance
- Combat readability refinements

Deliverables:
- Playtest insights deck
- Prioritized action plan

### Week 11 — Polish Sprint
- VFX polish and camera smoothing
- Animation timing polish
- Final narrative pacing trims

Deliverables:
- Candidate build RC-1
- Known issues list (must-fix/ship-risk)

### Week 12 — Greenlight Package
- Final bugfix + cert sanity checks
- Capture gameplay footage and KPI evidence
- Build publisher demo package

Deliverables:
- Vertical Slice Gold Candidate
- Executive review deck
- Publisher submission bundle

---
## Definition of Done (Vertical Slice)
- 90+ minutes stable playable content
- Boss fight complete with narrative integration
- Crash-free session > 99%
- Co-op satisfaction score >= 8.5 internal playtest
- Visual/audio quality accepted by Art/Audio Directors

## Staffing Rhythm
- Daily discipline standups
- 2x weekly cross-discipline sync
- Weekly milestone review with decision log

## Escalation Rules
- Red risk unresolved > 72h escalates to production director
- Scope additions require trade-off approval in same sprint
- Quality gate failure triggers immediate freeze on new features
